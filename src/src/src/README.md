# 🚀 Stellar Wallet App

## 📌 Description
A blockchain wallet app using Stellar network to create accounts, check balances, and send XLM.

## ⚙️ Setup
cd frontend  
npm install  
npm run dev  

## ✅ Features
- Create account  
- Check balance  
- Send XLM  

## 📸 Output
- Wallet created  
- Balance displayed  
- Transaction successful
